



## 1. 确定安装node.js

```bash
node -v
# -> v9.11.1
```

```bash
npm -v
# -> 6.4.1
```



## 2. Git

仓库地址:

https://github.com/ilexwg/qunar-travel.git



## 3. Vue脚手架安装

安装vue-cli

```bash
npm install -g @vue/cli
```

check

```bash
vue --version
# -> 2.9.6
```



## 4. 初始化项目

```bash
vue init webpack <project name>
```



## 5. 启动项目

```bash
npm run dev
```

在浏览器中输入:

http://localhost:8080
